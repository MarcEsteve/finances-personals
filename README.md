# Finances Personals 2026

Aplicació web per registrar i analitzar ingressos i despeses de forma simple i clara.

Inclou alta/eliminació de transaccions, dashboard de resum i informes visuals (línies i barres) amb filtres per any i mes.

## Stack tecnològic

- **Frontend:** React + TypeScript + Vite + Recharts
- **Backend:** Node.js + Express
- **Base de dades:** MongoDB
